###排序算法总结
|算法|时间复杂度|额外空间复杂度|稳定性
|---|---|---|---|
选择排序|O(N^2)   |O(1)   |无
冒泡排序|O(N^2)   |O(1)   |有
插入排序|O(N^2)   |O(1)   |有
归并排序|O(N*logN)|O(N)   |有
随机快排|O(N*logN)|O(logN)|无
堆排序 |O(N*logN)|O(1)   |无
计数排序|O(N)    |O(N)   |有
基数排序|O(N)    |O(N)   |有

###不基于比较的排序
桶排序思想下的排序：计数排序 和 基数排序
1. 桶排序思想下的排序都是不基于比较的排序
2. 时间复杂度是O(N)，额外空间复杂度是O(M)
3. 应用范围有限，需要样本的数据状况满足桶的划分
